# Documentation Directory

This directory contains detailed documentation for the RoCE PoC project.

## Documents

### Setup and Configuration
- **[setup-guide.md](./setup-guide.md)**: Comprehensive step-by-step guide for setting up the RoCE PoC environment in NVIDIA Air. Includes lab access, switch configuration, host configuration, validation, and testing procedures.
- **[pfc-configuration-guide.md](./pfc-configuration-guide.md)**: Practical step-by-step guide for configuring PFC on SONiC switches using config_db.json, with real lab examples and troubleshooting tips.

### Lab Information
- **[lab-topology.md](./lab-topology.md)**: Detailed topology documentation for the SONiC Numbered BGP EVPN VXLAN Demo lab. Includes device names, IP addressing, connectivity, VLAN/VNI mappings, and access credentials.

### Results and Analysis
- **[results.md](./results.md)**: Template for documenting test results, performance metrics, validation outcomes, and analysis. Use this to record your PoC findings.

## Quick Reference

**Starting the PoC?** Begin with:
1. Read the main [README.md](../README.md) for project overview
2. Review [lab-topology.md](./lab-topology.md) for lab-specific details
3. Follow [setup-guide.md](./setup-guide.md) for step-by-step instructions
4. Use [results.md](./results.md) to document your findings
